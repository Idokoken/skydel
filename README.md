#  Skydel Website

## Usage
This is a website for ordering food and ordering delivery Ride.

## Description

Skydel website is developed to provide seamless and user freindly food and parcel ordering and delivery experience. 

## Technology used

The technologies used for developing the backend includes
1. Java
2. Spring framework
3. MySql
4. some external dependencies like Spring Web, Spring Security, Spring Data Jpa, Lombok etc

The technologies used for developing the frontend includes
1. React(Javascript)
1. CSS
2. Bootstrap Framework
